## If you see errors in the terminal:

**Common issues:**

1. **Port 3000 already in use:**
   - Another app is using port 3000
   - Solution: Close that app or change port

2. **Module not found:**
   - Run: `npm install`

3. **Database connection error:**
   - Check `.env` file has correct DATABASE_URL
